## 快速开始

### 安装

下载对应平台的安装包：
- **Windows**: `InfoMagic_x.x.x_x64-setup.exe` 或 `InfoMagic_x.x.x_win64.zip`

### 首次使用流程

1. 配置 LLM API（用于生成大纲）
2. 配置图像生成 API（用于生成配图）
3. 创建新项目
4. 输入主题，生成大纲
5. 切分页面，生成图片
6. 导出 PDF 或保存图片

## 配置 API

### LLM 配置（文生文模型）

用于智能生成大纲内容。支持多种提供商：

| 提供商 | 端点 | 推荐模型 |
|--------|------|----------|
| OpenAI | `https://api.openai.com/v1` | gpt-4o, gpt-5 |
| DeepSeek | `https://api.deepseek.com/v1` | deepseek-chat, deepseek-v4 |
| 阿里云 DashScope | `https://dashscope.aliyuncs.com/compatible-mode/v1` | qwen-max, qwen-plus |
| 自定义 | 任意 OpenAI 兼容端点 | - |

**配置步骤：**
1. 点击左侧菜单「API 配置」
2. 填写 LLM 配置信息
3. 点击「测试连接」验证
4. 保存配置

### 图像生成配置

用于为页面生成配图。支持多种提供商：

| 提供商 | 端点 | 推荐模型 |
|--------|------|----------|
| OpenAI | `https://api.openai.com/v1/images/generations` | gpt-image-2 |
| Google Imagen | `https://generativelanguage.googleapis.com/v1beta/images` | imagen-3.0-generate-002 |
| 阿里云 DashScope | `https://dashscope.aliyuncs.com/api/v1/services/aigc/text2image/image-synthesis` | wanx-v1 |
| Agnes AI | `https://apihub.agnes-ai.com/v1/images/generations` | agnes-image-2.0-flash |
| 自定义 | 任意兼容端点 | - |

**提示：** 不同的图像生成模型有不同的风格特点，建议根据内容类型选择合适的模型。

## 创建项目

### 新建项目

1. 点击左侧菜单「项目列表」
2. 点击「新建项目」
3. 输入项目名称/主题

项目名称建议简洁明了，如：「产品发布会」「年度总结」「教程指南」等。

### 项目结构

每个项目包含以下文件：
```
projects/你的项目名/
├── outline.md      # 大纲文件
├── page-01.md      # 第1页内容
├── page-01.png     # 第1页图片
├── page-02.md      # 第2页内容
├── page-02.png     # 第2页图片
└── ...
```

## 生成与编辑大纲

### AI 生成大纲

1. 进入项目后，在大纲页面输入提示词
2. 提示词应该清晰描述你想要的内容

**提示词示例：**
```
制作一份关于人工智能发展趋势的信息图表，包含以下内容：
1. AI 发展历程
2. 当前主要技术方向
3. 未来应用场景
4. 行业挑战与机遇

风格：科技感、简洁、专业
目标受众：技术爱好者
```

### 生成模式

- **简单**：5-10 页，适合快速概览
- **中等**：10-20 页，适合一般演示
- **详细**：20-50 页，适合深度讲解
- **自定义**：自行指定页数

### 编辑大纲

生成后可以直接编辑 Markdown 内容：
- 使用 `---` 分隔不同页面
- 每页包含标题和内容
- 支持 Markdown 格式（加粗、列表、链接等）

### 讲稿备注

可以在页面内容中添加讲稿备注，生成图片时会自动过滤：

```markdown
# 演讲主题

内容描述...

**讲稿备注**：这里写演讲时的提示，不会出现在图片中
```

### 切分页面

大纲编辑完成后，点击「切分页面」将内容按 `---` 分隔符拆分为独立的页面文件。

## 页面编辑与图片生成

### 界面布局

- **左侧**：页面标签 + Markdown 编辑器
- **右侧**：图片预览 + 操作按钮

### 选择模板和风格

在工具栏中选择：
- **模板**：决定页面的整体布局风格（封面/内容/封底）
- **风格**：决定配色、字体等视觉元素
- **尺寸**：选择图片比例（16:9 横屏、9:16 竖屏等）

### 生成图片

**批量生成**：
- 点击「批量生成」一次性生成所有页面图片
- 支持并发处理（最多 3 个同时生成）
- 适合快速生成整套内容

**单页生成**：
- 点击「生成当前」仅生成当前页面
- 适合微调单个页面

**图片微调**：
- 生成图片后，点击「微调」按钮
- 输入修改要求，如「将背景改为蓝色」「增加一个图表」
- AI 会基于当前图片进行修改

### 页面类型识别

系统自动识别页面类型：
- **第 1 页**：封面页 → 使用 `front-cover.png` 模板
- **最后一页**：封底页 → 使用 `back-cover.png` 模板
- **中间页**：内容页 → 使用 `content.png` 模板

也可通过内容关键词识别：
- 包含「封面」「cover」→ 封面页
- 包含「封底」「谢谢」「感谢」→ 封底页

## 模板系统

### 什么是模板？

模板是一组参考图片，用于指导 AI 生成图片时的布局和风格。每个模板包含三张图片：
- `front-cover.png` - 封面模板
- `content.png` - 内容页模板
- `back-cover.png` - 封底模板

### 模板的作用

1. **保持一致性**：所有页面遵循相同的视觉风格
2. **控制布局**：Logo、标题区、内容区位置固定
3. **品牌识别**：嵌入公司 Logo 和品牌色

### 使用模板

1. 在「模板管理」创建新模板
2. 上传封面、内容页、封底的参考图片
3. 在页面编辑时选择该模板

### 不使用模板

选择「不使用模板」时，AI 将完全根据文字描述生成图片，适合创意性内容或不需要固定风格的场景。

## 风格系统

### 什么是风格？

风格是一段 Markdown 格式的描述文件，用于指导 AI 生成图片的视觉风格，包括配色、背景、字体等。

### 风格文件结构

```markdown
# 风格名称

## 风格锚点
简洁描述整体风格特点

## 配色
- 主色：深海蓝 (#1E3A5F)
- 强调色：科技青 (#00B8D9)
- 背景色：浅灰白 (#F5F5F5)

## 背景基调
浅灰白底色，可加入渐变效果

## 标题样式
大号粗体，深色，居中或左对齐

## 卡片样式
圆角卡片，轻微阴影，白色背景

## 图标样式
线性图标，主色调

## 线条样式
细线条，连接各元素

## 禁止规则
- 不要出现无关的公司名称或 Logo
- 不要使用干扰阅读的复杂背景
```

### 从图片提取风格

1. 在「风格管理」点击「提取风格」
2. 选择一张参考图片
3. AI 自动分析图片的配色和风格
4. 生成风格描述，可编辑后保存

**注意：** 提取风格需要配置支持多模态的 LLM（如 GPT-4o、Qwen-VL、DeepSeek-V4）。

### 创建自定义风格

1. 点击「新建风格」
2. 按照模板填写各项内容
3. 保存后即可在项目使用

## 最佳实践

### 提示词技巧

**好的提示词：**
- 清晰描述主题和目标受众
- 指定期望的内容结构
- 说明风格偏好

**示例：**
```
为一家科技公司制作年度总结信息图表，包含：
1. 公司简介（封面）
2. 年度里程碑（时间线展示）
3. 核心产品数据（图表展示）
4. 团队风采（照片墙风格）
5. 未来展望
6. 致谢（封底）

风格：科技感、蓝色主色调、简洁专业
目标受众：投资者和合作伙伴
```

### 模板使用建议

| 场景 | 建议 |
|------|------|
| 企业演示 | 使用公司模板，保持品牌一致性 |
| 产品介绍 | 创建产品专属模板，突出产品调性 |
| 个人创作 | 可不使用模板，发挥创意 |
| 系列内容 | 统一模板和风格，形成系列感 |

### 风格选择建议

| 内容类型 | 推荐风格 |
|----------|----------|
| 商务报告 | 深色主题、专业简洁 |
| 科技产品 | 科技感、蓝色/紫色系 |
| 教育培训 | 清新明亮、易读性强 |
| 活动宣传 | 活力色彩、吸引眼球 |

### 提高生成质量的技巧

1. **大纲要详细**：每个页面的描述越详细，生成效果越好
2. **选择合适的模板**：模板能显著提高一致性和专业度
3. **微调优化**：首次生成不满意时，使用微调功能调整
4. **多次尝试**：AI 生成有一定随机性，不满意可重新生成

## 常见问题

### Q: 支持哪些语言？

A: 支持中文和英文。建议使用中文提示词以获得更好的中文内容生成效果。

### Q: 图片生成失败怎么办？

A: 检查以下几点：
1. API Key 是否正确配置
2. API 余额是否充足
3. 网络连接是否正常
4. 查看错误日志了解详情

### Q: 如何让生成的图片风格更一致？

A: 
1. 使用统一的模板
2. 选择或创建明确的风格文件
3. 在大纲中使用一致的表达方式

### Q: 可以修改已生成的图片吗？

A: 可以。使用「微调」功能，输入修改要求即可基于当前图片进行修改。

### Q: 支持导出哪些格式？

A: 目前支持：
- PNG 图片（每页一张）
- PDF 文件（所有页面合并）

### Q: 数据存储在哪里？

A: 所有数据存储在本地：
- 配置文件：`config.yaml`
- 项目文件：`projects/` 目录
- 模板文件：`templates/` 目录
- 风格文件：`styles/` 目录

### Q: 如何查看错误日志？

A: 点击左侧菜单「日志」查看详细的操作日志和错误信息。
