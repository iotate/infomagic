# 科普百科风格

现代科普百科卡片风格，适合动植物、产品、知识主题等内容。

## 风格提示词

- 风格：浅色干净背景、柔和配色、轻阴影、精致小图标、圆角信息框、整洁排版、高信息密度但不拥挤
- 背景：浅色干净底色、柔和渐变、留白充足
- 文本：清晰标题层级、醒目标签、简洁短句、数字信息醒目
- 细节：圆角模块化信息分区、局部特征放大细节、可视化评分卡、Top 5 模块、精美小图标

## 核心要求

1. **主视觉**：清晰美题主视觉，居中突出
2. **模块布局**：多个圆角模块化信息分区，整齐清爽
3. **标题层级**：清晰的标题层级与重点标签
4. **百科内容**：简洁但信息丰富的百科内容

## 内容栏目（根据主题自动适配）

- 基础档案 / 分类信息
- 外观特征 / 习性生态
- 形成机制 / 结构组成
- 生长条件 / 使用条件
- 养护建议 / 维护建议
- 风险与注意事项
- 适合人群 / 适用场景
- 优缺点对比
- 快速评分卡

## 禁止规则

- 不要做成普通商业宣传海报
- 不要使用广告感过重的设计
- 重点突出"知识整理""模块信息""图鉴式展示"
- 避免长段正文，用简洁短句呈现
