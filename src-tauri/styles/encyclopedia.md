# 科学百科全书风格

经典百科全书插图风格，适合科普知识、人物传记、动植物图鉴等主题。

## 风格提示词

- 风格：复古泛黄纸张背景、精细科学插图风格、细腻线条、极致复杂专业、学术感、无品牌标识
- 背景：复古泛黄米色纸张质感、仿古纸纹、轻微做旧效果
- 文本：主标题使用中文书法体、正文使用清晰手写风格中文、所有注释必须清晰可读、简体中文
- 细节：精致解剖图、显微镜放大窗口、连接线标注、箭头指示、虚线框、圆角信息模块、6-8个知识模块环绕主体

## 核心要求

1. **中央主体**：3D立体效果，仿佛从纸面跃出，极高精度写实感
2. **布局**：中央留白突出主体，周围环绕6-8个结构化知识模块
3. **连接线**：精细引导线、箭头、括号连接主体与模块
4. **文字**：大量清晰中文手写注释，每个细节都要有标注

## 适用主题

- 人物：解剖结构、生理过程、历史时间线、主要贡献、认知过程、遗传/进化、全球影响
- 动物：外观素描与解剖、行为模式与生命周期、消化与骨骼系统、栖息地与分布地图、独特适应性、进化历史、共生关系、保护状态
- 植物：植物素描与解剖、光合作用与生命周期、细胞结构、药用价值、环境适应性、分布地图、遗传变异、历史用途

## 禁止规则

- 不要出现任何品牌Logo或标识
- 不要使用无法阅读的乱码文字
- 不要让中央主体被模块遮挡
- 保持信息密度但要有组织性
